58// Since we need to suport float, double, cx_float and cx_double,
59// as well as preserve the sanity of the user,
60// we cannot simply include all the SuperLU headers due to their messy state
61// (duplicate definitions, pollution of global namespace, bizarro defines).
62// As such we are forced to include only a subset of the headers
63// and manually specify a few SuperLU structures and function prototypes.
64//
65// CAVEAT:
66// This code requires SuperLU version 5.2,
67// and assumes that newer 5.x versions will have no API changes.
