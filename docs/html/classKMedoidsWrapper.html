Detailed Description

Python wrapper for KMedoids class.

KMedoidsWrapper class. Is the Python wrapper generated by Pybind that allows for calling the C++ code in Python.

Class implementation for running KMedoids methods.

KMedoids class. Creates a KMedoids object that can be used to find the medoids for a particular set of input data.

Parameters
n_medoidsNumber of medoids/clusters to create
algorithmAlgorithm used to find medoids; options are "BanditPAM" for the "Bandit-PAM" algorithm, or "naive" to use the naive method
verbosityVerbosity of the algorithm, 0 will have no log file emitted, 1 will emit a log file
max_iterThe maximum number of iterations the algorithm runs for
logFilenameThe name of the output log file
